🔗 Method 1: Direct Link

  1. Copy the URL from above
  2. Open browser on Android phone
  3. Paste URL in address bar
  4. Press Enter to load
  5. Tap Menu (⋮) → "Add to Home Screen"

📷 Method 2: QR Code

  1. Open Camera app on Android phone
  2. Point at QR code above
  3. Tap notification/link
  4. App opens in browser
  5. Tap Menu (⋮) → "Add to Home Screen"

⚙️ Method 3: APK Installation

  1. Build APK: ./build_android.sh
  2. Transfer to Android phone via USB
  3. Open file manager, find APK
  4. Tap to install
  5. Allow installation and complete
